# Mobile Detailing Business - 2025 Analytics Dashboard

This project is an interative Excell dashboard I built for my mobile detailing business. It helps me monitor performance and make better decisions using real data.

# Features 
Total revenue for the year
Most valuable service ( by revenue)
Most popular service ( by number of jobs)
Top-performing city
Peak month
Average sale per job
Payment method distribution
Interative filtering with slicers

# Tools and Skills
Microsoft Excel
Data cleaning and preprocessing
Pivot tables and pivot charts
Slicers and interative filtering
KPI creation
Dashboard design and formating
